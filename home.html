NIM-Z

SETUP

Please make sure that the following are installed correctly:

  • PHP
  • Myapache2 installed and running
  • SSH
  • Expect
Also, make sure that the NIM-Z folder is in /var/www/html folder and php has permissions to read and write into the files of the folder.

START GAME

Both Players need to select distinct options. Player 1 chooses the number of piles.
Value should be Min 3 and Max 16. Enter 0 for Random value.
Required for SCP purpose.

TUTORIAL

  • NIMZ is a mathematical game of strategy in game theory where two players take turns removing objects from distinct heaps (Row #1...Row #4).
  • On each turn, a player must remove at least one object, and may remove any number of objects provided they all come from the same heap - in this case a heap is a row.
  • The player to make the last move is the Winner.
  • The player who cannot make any valid move is the Loser.
  • Clicking on a block implies that the user wishes to remove all the blocks above it in the row.
  • Make sure that Player 2 selects 'play' after atleast 10 seconds of Player 1 selecting 'play'.
